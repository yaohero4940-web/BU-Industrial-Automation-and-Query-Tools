Formula check

Checks and refinements applied to the draft formulas

01

Psychrometric enthalpy unit fixed

When humidity ratio is displayed in g/kg dry air, enthalpy must use kg/kg: h = 1.005t + w(2501 + 1.86t), where w = d/1000.

02

Wet-bulb relation corrected

Uses Pv = Pws(twb) - A P (tdb - twb), where A = 0.00066(1 + 0.00115twb), with pressure in kPa.

03

Pipe friction factor completed

Laminar flow uses 64/Re; turbulent flow uses the Swamee-Jain explicit formula so lambda is computable.

04

Cooling tower blowdown clarified

Recommended balance: Qb = Qe/(N - 1) - Qwind. A conservative draft method is also shown for comparison.

05

Outdoor-air humidification converted

Msteam = m_oa x 3600 x (d_room - d_out) / 1000, avoiding a 1000x error from treating g as kg.

06

Result validity warnings added

Inline warnings cover negative temperature differences, N <= 1, Pv above saturation, invalid efficiency, and Reynolds transition ranges.
